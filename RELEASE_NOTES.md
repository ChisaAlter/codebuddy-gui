# CodeBuddy GUI 0.1.0

首个可安装预览版本，面向本地多项目 CodeBuddy CLI 工作流。

## 已提供

- 多项目和多对话持久化，支持并行 CodeBuddy 运行时与后台会话。
- 文件浏览、Monaco 编辑器、项目终端和 Git 工作流。
- 插件、Worker、任务、实例、日志、指标、链路和远程渠道管理。
- Windows 托盘、单实例唤回、窗口状态恢复和可选安装目录。
- 次级页面按需加载，编辑器仅在进入工作区后加载。
- 首次启动时若 CodeBuddy CLI 不可用或项目目录已移动，会显示可操作的明确错误信息。

## 使用要求

- Windows x64。
- 本机可直接运行 `codebuddy` CLI，并已完成其必要配置。

## 数据保留

- 覆盖安装会保留现有项目和对话数据。卸载程序默认保留用户数据，方便后续重装恢复。
- 产品状态保留最近一次有效备份；主状态文件损坏时自动恢复备份，并保留原始坏文件用于诊断。
